# Banking Management System

This project is a database-driven Banking Management System using MySQL and Python.

## 1. Project Description

The Banking Management System is designed to manage customers, bank accounts, transactions, employees, and branches. The system supports basic banking operations such as opening accounts, depositing money, withdrawing money, transferring money, and generating reports.

## 2. Technologies Used

- MySQL
- MySQL Workbench
- Python
- mysql-connector-python
- GitHub

## 3. Main Features

- Customer management
- Account management
- Deposit, withdrawal, and transfer transactions
- Transaction history
- Employee and branch management
- Customer balance report
- Transaction summary report
- Indexes, views, stored procedures, functions, and triggers
- User roles and access control
- Database backup and recovery
